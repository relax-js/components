# @relax.js/components

Pre-built Web Components for [Relax.js](https://github.com/relax-js/core). Table, TreeView, Tabs, Menu, and Spinner ready to use.

## Install

```bash
npm install @relax.js/components
```

Requires [`@relax.js/core`](https://github.com/relax-js/core) as a peer dependency.

## Components

| Component | Tag | Description |
|-----------|-----|-------------|
| [Table](docs/Table.md) | `<r-table>` | Data table with declarative columns and pagination |
| [TreeView](docs/TreeView.md) | `<r-treeview>` | Hierarchical tree with expand/collapse and search |
| [TabsContainer](docs/TabsContainer.md) | `<r-tabs>` | Tab panels for organizing content |
| [TopMenu](docs/TopMenu.md) | `<r-top-menu>` | Horizontal navigation menu bar |
| [Spinner](docs/Spinner.md) | `<r-spinner>` | Loading overlay with customizable spinner |

## Usage

```typescript
import { Table, TreeView, TopMenu } from '@relax.js/components';
```

See the [docs](docs/Components.md) for full API documentation.
